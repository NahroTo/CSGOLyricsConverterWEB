Description
This app is able to turn the lyrics of your favorite song into a CSGO script.
The CSGO script lets you shout in chat every line of the song one by one with one bindkey.

Instructions

  1. Follow the steps.
  2. Download the result, and put the file in your CSGO cfg folder.
  3. Finally, when in-game, type in the console
    exec <filename>
  4. Now press the bindkey you chose and you'll see the action yourself.
Happy singing!
